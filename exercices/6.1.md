# Exercice 6.1 : *Mongoose*

## Warning

- Supprimez l'id numérique des informations à stocker dans la base de données car une nouvelle clé *_id* contenant la clé est générée automatiquement par MongoDB afin d'identifier une entité de manière unique.

## Enoncé

1. Reprenez [5.3](./5.3.md), toujours en respectant les recommandations de l'API REST, faites les actions suivantes dans la base de données à partir des schemas et modèles de Mongoose sur les entités (teams, games, etc.) : 
- Ajoutez une équipe
- Modifiez une équipe
- Supprimez une équipe
