# Exercice 5.3 : mise en place d'une *API* complète

1. Créez un nouveau projet avec Express **sans** ***CLI*** (avec *npm*)
2. Créez une ***API REST*** pour la **Ligue des champions 2025-2026** pour répondre aux besoins ci-dessous, vous pouvez vous appuyer sur les informations du site [***EuroSport***](https://www.eurosport.fr/football/ligue-des-champions/classements.shtml) sur la compétition. Votre application écoute sur le port ***8053***

**PS : il n'est pas demandé de restituer tous les clubs et matchs, un pétit échantillon suffira amplement** 


| Méthode HTTP | Endpoint   | Ressources             |
| -------- | -------------- | ---------------------- |
| `GET`    | `/teams`       | Liste des équipes (clubs) |
| `GET`    | `/teams/:name`   | Un seul club |
| `POST`   | `/teams` | Ajouter une équipe       |
| `GET`    | `/games` | Liste de tous les matchs |
| `POST`   | `/games` | Ajouter un match       |
| `PUT`    | `/games/:id` | Mettre à jour un match |

- Pour la route ***POST /teams*** : il faut stocker les utilisateurs dans un fichier JSON.
- Pour simplifier : à chaque ajout d'une équipe, récupérez le contenu de votre fichier, puis ajoutez le nouvelle équipe dans votre tableau puis insérez les nouvelles données dans le JSON en écrasant le contenu initial.
- Pour parser le JSON (transformer le contenu textuel de votre fichier en objet JS), vous utilisez la fonction ***JSON.parse(data)*** avec data ici "un string en JSON" contenu fichier JSON.


## La structure des données de votre API

```json
{
  "teams": [
    { "id": 1, "name": "Bayern Munich", "country": "Germany" },
    { "id": 2, "name": "Arsenal", "country": "UK" },
    { "id": 3, "name": "Paris Saint-Germain", "country": "France" },
    { "id": 4, "name": "Athletico de Madrid", "country": "Spain" }
  ],
  "games": [
    { "id": 1, "home": 3, "away": 1, "score": "5-4", "date": "2026-04-28" },
    { "id": 2, "home": 4, "away": 2, "score": "1-1", "date": "2026-04-29" }
  ]
}
```
Ici les propriétés *home* et *away* stockent les identifiants des équipes par exemple pour le match avec l'*id* 1, c'est le *Bayern* qui a joué à l'extérieur avec une défaite 5-4 face à la *Paris Saint-Germain* (équipe à domicile).
