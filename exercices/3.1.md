# Exercice 3.1 : lecture/écriture

1. Depuis votre programme Node, créez un nouveau fichier  *index.html* et y ajouter le contenu ci-après :
```html
<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Lecture/écriture avec Node</title>
</head>
<body>
  <h1>Module fs</h1>
</body>
</html>
```
2. Lisez le contenu de ce fichier et affichez-le dans la *console*.

---

## Bonus

1. Servez le HTML dans un serveur Web HTTP
2. Transformez Les promesses que vous avez utilisé pour writeFile ou readFile en callback avec callbackify de node:util ou transformez les callbacks de writeFile ou readFile en promesses avec promisify de node:util
