# Variables environnement

Plusieurs méthodes pour utiliser des variables d'environnement avec NODE cf. [article](https://nodejs.org/learn/command-line/how-to-read-environment-variables-from-nodejs) :

1. Au lancement avec l'option ***--env-file***
2. Dans un programme (fichier Node), via l'utilisation de la fonction ***loadEnvFile()*** de ***node:process***
